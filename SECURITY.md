# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | ✅ Active development |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately.

**Do not** open a public issue.

Send details to **kyawzayartun.contact@gmail.com**.

Please include:

- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Potential impact

You should receive a response within 48 hours.

## Security Best Practices

This plugin follows these security principles:

- All file paths are validated before access
- External input is never trusted without validation
- Arbitrary file access is prevented
- Unsafe permissions are not requested
- User privacy is protected
- No secrets or keys are logged or committed
